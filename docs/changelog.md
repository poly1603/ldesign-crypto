# 更新日志

## v1.0.0 (2024-01-XX)

### ✨ 新功能

#### 🔐 核心加密功能
- **对称加密**: 完整支持 AES、DES、3DES 算法
  - 支持多种加密模式：ECB、CBC、CFB、OFB、CTR、GCM
  - 支持多种填充方式：PKCS7、PKCS5、ZeroPadding、NoPadding
  - 自动 IV 生成和管理
- **非对称加密**: 完整支持 RSA、ECC 算法
  - RSA 密钥生成（1024/2048/4096位）
  - RSA 加密解密和数字签名
  - ECC 椭圆曲线加密支持
- **哈希算法**: 完整支持主流哈希算法
  - MD5、SHA-1、SHA-256、SHA-512
  - HMAC 消息认证码
  - PBKDF2 密钥派生函数
  - 文件哈希计算（分块处理）
  - 密码强度检测和生成

#### 🇨🇳 国密算法支持
- **SM2**: 椭圆曲线公钥密码算法
  - 密钥生成、加密解密
  - 数字签名和验证
  - 密钥协商协议
- **SM3**: 密码杂凑算法
  - 256位哈希输出
  - HMAC-SM3 支持
- **SM4**: 分组密码算法
  - ECB/CBC 加密模式
  - 128位密钥支持

#### 🎨 Vue 3 集成
- **组合式 API**: 完整的 Composables 支持
  - `useSymmetricCrypto` - 对称加密
  - `useAsymmetricCrypto` - 非对称加密
  - `useHash` - 哈希算法
  - `useSM` - 国密算法
- **响应式状态**: 自动管理加载状态、错误信息、操作结果
- **插件安装**: 支持 `app.use()` 全局安装

#### 🚀 高级特性
- **性能监控**: 实时监控加密操作性能
  - 执行时间统计
  - 操作次数统计
  - 性能阈值告警
- **智能缓存**: LRU 缓存策略
  - 自动缓存计算结果
  - 可配置缓存大小和过期时间
  - 缓存命中率统计
- **插件架构**: 模块化设计
  - 支持自定义算法插件
  - 动态插件加载和卸载
  - 插件生命周期管理
- **错误处理**: 完善的错误处理机制
  - 分类错误类型
  - 详细错误信息
  - 错误恢复建议

#### 🛠️ 工具功能
- **随机数生成**: 多种字符集支持
- **密钥生成**: 自动生成强密钥
- **密钥管理**: 密钥验证、格式转换
- **编码转换**: 支持多种编码格式
